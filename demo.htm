Checkboxy demo page

Usage

Usage is pretty simple. First, include the jQuery library

<script src="http://code.jquery.com/jquery-1.10.2.min.js"></script>

Then, include CSS and JS files of checkboxy

<link rel="stylesheet" href="jquery.checkboxy.css" type="text/css">
<script src="jquery.checkboxy.js"></script>

And the final step is to connect checkboxy plugin to elements on a page

$(function() {
  $('input[type="checkbox"]').checkboxy();
});

That's it!
